Sensorial, sensor data at your hand!

This is a free API that can be used to collect the following features

  • Sattelite from Hubble and ISS
  • Solar System data (planets and the moon)
  • Seasons data
  • Earthquake data

Please check the Documentation section on the left to better explanation of usage.

Sources

Hubble and ISS data are taken from Celestrak.

  • Hubble Space Telescope (HST)
  • International Space Station (ISS)

Earthquake Data is taken from USGS.

  • USGS Earthquake (all hour)

Solar System and Seasons are calculated using pyephem.

Github

This site and all it's source code are available in Github at rdenadai/sensorial.
